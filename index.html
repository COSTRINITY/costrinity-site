Quantitative Core

THE MATHEMATICS OF THE TOP 0.4%

SNiPr v5 is built on institutional-grade quant research — the same mathematical foundations used by the world's top algorithmic trading firms, adapted for Solana's bonding curves.

Position Sizing
Kelly Criterion
f* = (b·p − q) / b
f* = optimal bet fraction b = avg_win / avg_loss p = win probability

Every position is sized dynamically based on historical edge — not a fixed amount. High-confidence trades get more capital. Low-confidence trades get less. The math maximizes geometric growth while surviving losing streaks.

Entry Selection
Bayesian Inference
P(A|B) = P(B|A)·P(A) / P(B)
Prior: 5% base rate Signals: dev buy, OBI, whales Gate: P > 18% + EV > 0

Every new token gets a posterior probability update using Bayes' theorem. Log-odds from 8 independent signals are combined. Only tokens above the probability threshold and with positive expected value are entered.

Volatility Regime
GARCH(1,1)
σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1}
LOW / MED / HIGH / MELT α = 0.15 (shock weight) β = 0.80 (persistence)

Generalized Autoregressive Conditional Heteroskedasticity — the same model used by quant hedge funds. SNiPr detects whether the market is in a calm or explosive regime and adjusts stops, sizing, and entry thresholds accordingly.

Market Microstructure
Order Flow Imbalance
OBI = (V_buy − V_sell) / (V_buy + V_sell)
Range: −1 (all sells) → +1 (all buys) Entry: OBI > +0.30 Exit: OBI < −0.20

The same signal used by HFT firms to predict 50–500ms price movements. OBI quantifies buy vs sell pressure in real time. SNiPr enters when buy pressure dominates and exits when distribution begins — before price reflects it.

Fat-Tail Risk
Cornish-Fisher VaR
z_CF = z + (z²−1)·s/6 + (z³−3z)·k/24
s = skewness (memecoins: −1.5) k = excess kurtosis (~8) Beats normal VaR by 30–50%

Standard stop losses assume Gaussian returns. Memecoins have fat tails — extreme moves happen far more often than any normal model predicts. Cornish-Fisher expands the quantile calculation to account for skewness and kurtosis, giving more accurate stop levels.

Survival Analysis
Gompertz Hazard Rate
P(dump) = 1 − e^{−λ₀(e^{βt} − 1)/β}
Median hold: 100 seconds P(dump @ 5 min) = 87.6% Exit targets decay with time

Adapted from biological survival analysis. The probability of a token dumping increases exponentially with time. SNiPr calculates this in real time and reduces take-profit targets accordingly — aiming for 2x in the first minute, not waiting for a 10x that statistically never comes.

Risk Assessment
Monte Carlo Simulation
RoR = ((1−edge)/(1+edge))^{B/r}
1,000 trade sequences simulated Halt if RoR > 35% Reduce if RoR > 15%

Before every session, SNiPr simulates 1,000 possible sequences of trades using historical win rate, average win, and average loss. If the risk of ruin exceeds safe thresholds, position sizes are automatically reduced. Capital preservation is mathematical — not guesswork.

Smart Money
Whale Convergence
LR = 2^N (per independent whale)
Threshold: > 0.3 SOL per buy 3 whales → 8× probability boost Independence verified by address

When multiple independent large wallets buy the same token within 30 seconds, it signals coordinated conviction from sophisticated traders. Each independent whale buy multiplies the Bayesian probability — because smart money clustering is a measurable on-chain alpha signal.

Performance Engine
Sortino + Calmar
Sortino = R̄ / σ_downside > 2.0
Sortino: only penalizes losses Calmar = CAGR / MaxDrawdown Circuit breaker if Calmar < 0.5

Unlike the Sharpe ratio (which penalizes upside volatility), the Sortino ratio only measures harmful risk. Calmar monitors whether gains outpace drawdowns. If performance degrades below thresholds, SNiPr automatically pauses and alerts — protecting capital in real time.
