COS TRINITY Docs 🔱

APIs and SDKs for integrating with SNiPr and VIGIL. Build autonomous trading agents, monitor on-chain activity, and plug into the COSTRINITY stack.

SNiPr APIhttps://api.costrinity.xyz
VIGILhttps://vigil.costrinity.xyz

Authentication

SNiPr API endpoints that require auth use an x-api-key header. VIGIL ingest uses x-vigil-key (your agent API key from the VIGIL dashboard).

# SNiPr API
curl -H "x-api-key: your_snipr_key" https://api.costrinity.xyz/api/snipe

# VIGIL ingest
curl -H "x-vigil-key: your_vigil_agent_key" https://vigil.costrinity.xyz/api/ingest

POST /api/snipe

Evaluate a Solana token mint. Returns an AI score (0–100) and a recommendation.

Requires x-api-key
POST https://api.costrinity.xyz/api/snipe
Fetches DexScreener data, runs heuristic AI scoring, returns buy/watch/skip recommendation.

Request Body

FieldTypeRequiredDescription
mintstring✓Solana token mint address (base58)
reasonstring—Optional context for the evaluation
max_solnumber—Max SOL to use if buying (informational)

Example

curl -X POST https://api.costrinity.xyz/api/snipe \
  -H "Content-Type: application/json" \
  -H "x-api-key: your_key" \
  -d '{"mint":"DezXAZ8z7PnrnRJjz3wXBoRgixCa6xjnB7YaB1pPB263"}'

Response

{
  "status": "evaluated",
  "mint": "DezXAZ8z7PnrnRJjz3wXBoRgixCa6xjnB7YaB1pPB263",
  "symbol": "BONK",
  "price_usd": 0.0000284,
  "volume_24h": 142500,
  "liquidity_usd": 95000,
  "ai_score": 84,
  "recommendation": "buy",
  "reason": "High volume + strong liquidity + positive momentum"
}

Rate Limits

10 requests per IP per minute. Returns 429 when exceeded.

VIGIL SDK

The official JavaScript/TypeScript SDK for logging agent events to VIGIL.

npm install @costrinity/vigil-sdk

Quick Start

import { createVigil } from '@costrinity/vigil-sdk';

const vigil = createVigil({ apiKey: 'your-vigil-agent-key' });

await vigil.info('Agent started');
await vigil.buy({ mint: 'DezX...', symbol: 'BONK', amount_sol: 0.05, tx: 'abc...' });
await vigil.sell({ mint: 'DezX...', symbol: 'BONK', proceeds_sol: 0.12, pnl_sol: 0.07, pnl_pct: 142 });
await vigil.error('RPC timeout', { endpoint: 'mainnet-beta' });

SDK Methods

MethodDescription
vigil.info(msg, payload?)Log an info/ok event
vigil.warn(msg, payload?)Log a warning
vigil.error(msg, payload?)Log an error
vigil.buy(opts)Log a buy — mint, symbol, amount_sol, tx, score, reason
vigil.sell(opts)Log a sell — mint, symbol, proceeds_sol, pnl_sol, pnl_pct, tx, reason
vigil.decision(msg, reasoning, payload?)Log an agent decision with reasoning
vigil.trace(name, fn, payload?)Wrap a function — auto-logs start, success, duration, errors
vigil.log(event)Raw event (full control over all fields)

POST /api/ingest (raw)

Direct REST API for non-JS environments. Auth via x-vigil-key.

Requires x-vigil-key 
curl -X POST https://vigil.costrinity.xyz/api/ingest \
  -H "Content-Type: application/json" \
  -H "x-vigil-key: your_agent_key" \
  -d '{
    "event_type": "buy",
    "status": "ok",
    "message": "Buy BONK — 0.05 SOL",
    "cost_sol": 0.05,
    "payload": { "mint": "DezX...", "tx": "abc..." }
  }'

Event Fields

FieldTypeDescription
event_typestringbuy · sell · error · decision · info · custom
statusstringok · warning · error
messagestringHuman-readable summary
payloadobjectAny JSON — your agent's data
cost_solnumberSOL spent (optional, for cost tracking)
cost_usdnumberUSD equivalent (optional)

ElizaOS Plugin

Let any ElizaOS agent snipe tokens via natural language commands.

npm install @costrinity/snipr-elizaos-plugin
import { sniprPlugin } from '@costrinity/snipr-elizaos-plugin';

// Add to your ElizaOS agent config
export default {
  plugins: [sniprPlugin],
  settings: {
    SNIPR_API_KEY: process.env.SNIPR_API_KEY,
    SNIPR_WALLET: process.env.SNIPR_WALLET,
  }
};

Available Actions

ActionAliasesWhat it does
SNIPE_TOKENSNIPE, BUY_TOKENEvaluate + optionally snipe a mint address
CHECK_SNIPR_BALANCESNIPR_BALANCECheck configured wallet SOL balance
GET_POSITIONSOPEN_TRADESList current open positions
